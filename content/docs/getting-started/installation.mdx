## Prerequisites

- **Rust** 1.75+ ([rustup](https://rustup.rs/))

## Build Tish (compiler/runtime)

Tish lives in the [tish](https://github.com/.../tish) repository:

```bash
git clone <tish-repo-url>
cd tish
cargo build --release -p tishlang
```

The binary is at `target/release/tish`.

### Editor tooling (optional, separate binaries)

Formatting and linting are **not** part of `tish` — same idea as rustfmt/clippy vs rustc:

```bash
cargo build --release -p tishlang_fmt    # → tish-fmt
cargo build --release -p tishlang_lint   # → tish-lint
cargo build --release -p tishlang_lsp    # → tish-lsp
```

Add the binaries you need to your `PATH`. See [Editor and IDE setup](/getting-started/editor/).

## Verify Installation

```bash
tish --help
tish run --help
tish compile --help
tish-fmt --help
tish-lint --help
tish-lsp --help
```

## zectre CLI (optional)

For deploying to the Zectre Platform, use the [zectre](https://github.com/.../zectre) CLI. Run from source (no install):

```bash
cd zectre
zectre --help
```
