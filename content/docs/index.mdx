## Quick Start

```bash
# Run with interpreter
tish run src/main.tish

# Run piped source (stdin), like node - / bun
echo 'console.log("hi")' | tish

# Compile to native binary
tish compile src/main.tish -o app
./app
```

## What is Tish?

Tish is a minimal language built in Rust that:
- **Looks like JavaScript** — familiar syntax with `let`, `const`, `fn`, arrows, template literals
- **Runs two ways** — interpret for fast iteration, compile to native for production
- **Is secure by default** — network, filesystem, and process APIs are feature-gated
- **Has no `undefined`** — uses `null` only; strict equality (`===`/`!==`) only
