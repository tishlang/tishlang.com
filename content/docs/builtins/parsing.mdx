title: Parsing
description: Parsing and type-check builtins in Tish.
---

## Functions

| Function | Description |
|----------|-------------|
| `parseInt(s, radix?)` | Parse string to integer. Optional radix (default 10). |
| `parseFloat(s)` | Parse string to number |
| `isFinite(v)` | True if v is finite number |
| `isNaN(v)` | True if v is NaN |

## Globals

| Global | Description |
|--------|-------------|
| `Infinity` | Positive infinity |
| `NaN` | Not-a-number |
